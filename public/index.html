Próximos Passos

  • Conectar ao Firebase
    Execute firebase login para autenticar sua conta Google
  • Criar Projeto no Console
    Acesse console.firebase.google.com e crie um novo projeto
  • Atualizar Project ID
    Edite o arquivo .firebaserc com o ID do seu projeto
  • Instalar Dependências
    cd functions && npm install
  • Configurar Gemini API Key
    Crie o secret no Google Cloud: gcloud secrets create GEMINI_API_KEY
  • Deploy
    firebase deploy

Sobre o Projeto

O TotalQuality é uma plataforma de análise de qualidade empresarial que utiliza o poder do Gemini 1.5 Pro para processar vídeos e fornecer insights detalhados sobre conformidade, métricas de qualidade e recomendações de melhoria. Com arquitetura multi-tenant, cada empresa mantém seus dados completamente isolados e seguros.
